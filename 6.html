Web technology - I

Introduction to the Internet

The Internet is the global network of networks, a huge collection of computers all over the world that are interconnected in various ways to form a web-net.

Components Required for Internet Connections

The basic requirements for the Internet are:

  • A TCP/IP (Transmission Control Protocol/Internet Protocol) enabled computer with a web browser.
  • An account with an ISP (Internet Service Provider).
  • A telephone line.
  • A MODEM to connect to the telephone line and computer.

Set of Computer Requirements:

  • Minimum 386 microprocessor chip with 16 MB RAM.
  • Color monitor with 256-color support and 640x480 resolution.
  • A hard disk with 200 MB of free space.
  • Multimedia sound card and speakers.

Internet Services and Applications

  • World Wide Web (WWW): A series of services interconnected through hypertext, enabling data exchange via the HTTP protocol. Created in 1990 at the European Particle Physics Laboratory in Geneva, Switzerland.
  • Web Browser: A computer program that accesses web pages and displays them. Examples include Internet Explorer, Safari, Mozilla Firefox, Opera, Netscape Navigator, etc.
  • FTP - File Transfer Protocol: An Internet tool that allows users to transfer files between computers. Files can contain data, graphics, text, etc.
  • Electronic Mail (E-mail): The most widely used feature on the Internet for sending and receiving messages. E-mail messages may include text, graphics, video, and audio.
  • Voicemail: A service that records audio messages and allows users to communicate with others through recorded voice.
  • USENET – View and News: A Bulletin Board Service (BBS) for the Internet that organizes messages into thousands of news groups covering specific areas of interest.
  • Chat: A real-time communication method where users can interact by typing. Users can join chat rooms and exchange information instantly.
  • IRC – Internet Relay Chat: A multi-user live chat facility with servers around the world. Users can create channels where all participants can see each other's messages.
  • Video Conferencing: A communication service that allows face-to-face conversations through video and audio, enabling real-time visual communications.
  • Universal Resource Locator (URL): A unique address used to identify locations on the Internet. Example format: 
    http://www.animalplanet.com
  • E-Fax (Electronic Fax): Allows sending and receiving faxes directly on the computer through the Internet. Messages are sent via the Internet server, unlike E-mail, which goes to a local file server.

    • Difference Between E-mail and E-Fax:

    Message first goes to the local server. Message first goes to the internet server and then to its destination.
    Message is stored in the hard disk of the destination computer. Message directly goes to print on the printer of the destination computer.
  • TELNET – Remote Login: Allows users to log in to a remote host computer from a local computer, enabling them to work on another computer over a network.
  • Navigators – Information Servers: Special features of the Internet to discover, locate, and retrieve information on various objects. Popular utilities include Gopher, Archie, Hytelnet, and WAIS.

Internet Terminologies

  • ISP (Internet Service Provider): A company that provides access to the Internet, such as Mercantile, Everest net, or World link in Nepal.
  • Link: A quick route to jump to other web pages. Links are typically colored differently from the rest of the text.
  • Search Engine: Software that helps search websites, such as Yahoo, Excite, Lycos, and Alta Vista.
  • Domain Name and IP (Internet Protocol) Address: The unique name that identifies an Internet site. The domain name is associated with an IP address, which establishes logical connections between computers on the network.
  • DNS Server: The server that resolves domain names to IP addresses, enabling internet communication.
  • Web Server: A server that hosts web pages and responds to requests from web browsers.
  • Usernet Newsgroups or Net News: Headings that categorize related topics and allow easy searching for specific subjects.
  • Mailing Lists: E-mail lists sent to individuals who have subscribed to them.
  • POP Server: A server where incoming e-mails are stored in a mailbox for users to access. It uses the Post Office Protocol to retrieve mail.
  • SMTP Server: A server used for transferring e-mail between computers through the Simple Mail Transfer Protocol.
  • FTP (File Transfer Protocol): A protocol that allows for transferring files between computers on the Internet.
  • Anonymous FTP: A facility for retrieving archived sites containing software, documents, images, and more without revealing your identity.
  • HTTP (Hypertext Transfer Protocol): A protocol used by web browsers and servers to communicate and transfer HTML documents.
  • Proxy Server: A server that improves access speed for a particular browser and helps update information on a site automatically.
  • Internet Roaming: Allows users to access their Internet accounts from anywhere in the world if their ISP is part of a global roaming community.

Introduction to HTML

HTML stands for Hyper Text Markup Language, which is used to create web pages that are used on the internet to exchange data and information. An HTML file itself is a text file that contains several tags and attributes. HTML was developed by Tim Berners-Lee and is based on SGML (Standard Generalized Markup Language). The tags and attributes used to create an HTML file are displayed using a web browser. Web browsers help understand all the tags and attributes given by the user and generate the content of the page. The different versions of HTML are: HTML, HTML+, HTML 1.0, HTML 2.0, HTML 3.2, HTML 4.01, and now HTML 5.0.

Use of HTML

  • It is used to create a basic layout or design of a web page.
  • Without HTML, the worldwide web wouldn't exist.
  • It allows features like embedding images, videos, and audio.
  • It allows users to customize and format characters, such as bold, italic, and underline.
  • It helps create links and lists.

Advantages of HTML

  • It is easy to use, flexible, and user-friendly.
  • It is supported by all web browsers and operating systems, such as Macintosh, Windows, and Unix.
  • It is easier to obtain and modify any content.
  • It is compatible with all search engines.
  • There are several new and advanced features in HTML 5.0.
  • HTML files take a short period of time to load.
  • Several tutorials can be easily accessed regarding HTML.

Disadvantages of HTML

  • It is difficult to create attractive and interactive web pages using only HTML (PHP, JavaScript, SQL, etc. are needed).
  • It cannot be used to create dynamic web pages.
  • It is only a scripting language and cannot be used as a programming language.
  • There are many incompatibilities in HTML.
  • There is no complete and universally accepted standard of HTML.

Tags

HTML tags are the instructions or commands given to the web browser in order to organize text, images, and any other components of web pages. These HTML tags are executed by the web browser accordingly. Like any programming language, HTML tags also help to perform specific actions. HTML tags are written inside angular brackets <>. There are two types of HTML tags:

HTML Tags and Attributes

Pair Tags

Pair tags are those which have both opening and closing tags. These are also known as container tags. If a tag is written only within angular brackets <>, it is the opening tag. The same tag, preceded by a slash (/) within angular brackets, is the closing tag.

Singular Tags

Singular tags are those that have only an opening tag but no closing tag. They are also known as empty tags.

Attributes

Attributes are descriptive properties of HTML tags written within the respective tag. They help to add extra features and allow the designer to customize and modify several components of web pages according to the requirement.

For example, for the <body> tag, some common attributes are:

  • bgcolor – Helps to add a background color.
  • text – Changes the color of the text in the webpage.
  • background – Adds an image in the background of the webpage.

Steps to Write and Execute an HTML Document

  1. Write all the HTML tags, attributes, and related information in a notepad.
  2. Save your document using the desired name followed by the .html extension (e.g., sample.html).
  3. Execute the HTML file by using any web browser.

Some Common Terminologies in Web Designing

Webpage

A webpage is generally an HTML document or written in any other scripting language that helps to execute information in the form of text, images, or other components through the internet.

Website

A website is a collection of interrelated webpages. It may contain any piece of information or may have information that is interrelated with several webpages connected through hyperlinks. A website has two parts:

  • Domain Name: The name used to access the website.
  • Web Server: The storage location where all information associated with a website is stored.

Home Page

The home page is the introductory page that is displayed first when we access the website.

Web Browser

A web browser is an application program that helps to access any content of webpages, which may be in any form using the HTTP (Hypertext Transfer Protocol). This protocol helps make communication between the web browser and web server. Examples include Google Chrome, Internet Explorer (Microsoft Edge), Safari, Mozilla Firefox, Netscape Navigator, etc.

URL

URL stands for Uniform Resource Locator. It is a standard way of referencing a protocol, host, and a directory. For example: https://readersnepal.com.

DBMS (Database Management System)

Data and Information

Data: Data are the raw facts and figures. Data are isolated, uninterpreted, and don't provide any meaning unless processed. For example, Ram, 19, 12, Pokhara.

Information: Information is the meaningful result obtained after processing data. It is generally a processed version of data. For example, "Ram is a student whose age is 19, and he lives in Pokhara-12."

Database

A database is a systematically organized collection of data that allows retrieval of information quickly and easily. It converts data into meaningful information. For example, a dictionary, mark ledger, census, and telephone directory are databases.

DBMS (Database Management System)

A DBMS is a software system that helps to maintain a database. It allows systematic and scientific organization and collection of data so that information can be retrieved quickly. Examples include MS Access, MySQL, Oracle, FoxPro, and dBase.

Advantages/Features/Importance of DBMS

  • It allows users to search data quickly and easily.
  • It helps reduce data redundancy (repetition).
  • It allows users to share data over a network.
  • It helps in data security and privacy.
  • It improves data consistency.
  • It provides an organized way of storing data.

Disadvantages of DBMS

  • Installation and operating costs are higher.
  • It requires more disk space.
  • It may sometimes have security and privacy issues.
  • Technical manpower (DBA) is required to maintain and operate the system.
