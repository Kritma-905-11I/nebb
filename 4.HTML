MS-Word (Word Processor)

Introduction:

Ms-Word is a text editing or word processing software develop by Microsoft Corporation, USA under MS-Office package. Word processing refers to use of computer to create, edit and print documents. A word processor enables us to create a document, store it electronically in a disk, display it on a screen, modify it by entering commands and characters from keyboard and print it on a printer. MS-Word is a windows based application it has many powerful and advance feature to manipulate the document, With the help of this program we can create any types of documents like letters, reports, thesis, manuals, brochures, advertisement an many other document. Beside creating, editing and modifying the documents MS-Word provides some special facilities such as checking spelling and grammatical error, to search for synonyms and antonyms, to work with columns, tables, special symbols, pictures, graphs, charts and many more. The document created with MS-Word have “.doc” extension. The newer version of MS-Word has newer extension “.docx”".

Features of MS-Word

  • Fast Operation: Since MS-Word is a digital text editing software, there is no mechanical movement associated with it, making it faster in operation.
  • Editing Text Feature (Character Formatting): Operations such as insert, delete, modify, undo/redo, and find/replace can be easily performed. The document can be customized with different styles like bold, italic, underline, font size, color, and font style.
  • Permanent Storage: MS-Word allows you to electronically store files and documents on a disk or any other storage media, making it easy to retrieve them when needed.
  • Graphics: MS-Word allows you to insert various objects such as pictures, graphs, and other media into your document as required.
  • OLE (Object Linking and Embedding): OLE is a technology used to share information between programs through objects. Objects include charts, equations, pictures, audio, and video files.
  • Mail Merge: Mail Merge is a useful tool for producing multiple letters, labels, envelopes, and more using information stored in a list, database, or spreadsheet.
  • Searching: MS-Word allows you to search for a specific word or text and replace or delete it with other necessary text throughout the entire document.
  • Printing: MS-Word allows users to print the documents created anytime and anywhere.
  • Formatting a Document: Word offers tools to format a document, such as changing paragraph alignment, line spacing, creating bullets and numbering, adding borders, shading, and watermarks to enhance the appearance and readability of a document.
  • Spelling and Grammar Checker: MS-Word provides tools to check for spelling and grammar errors, notifying users with colored wavy lines (red for spelling errors and green for grammar errors).
  • Use of Thesaurus: MS-Word’s thesaurus feature allows users to look up synonyms and antonyms for words with a few clicks of the mouse, and even replace words with their respective synonyms or antonyms automatically.

Cell Reference in Excel (V. imp)

A cell location in a spreadsheet is referred to as its cell reference. To find a cell reference, simply look at the column heading to find which column the cell is in and across to find which row it is in. The cell reference is a combination of column letter and row number such as A1, B3, Z9. When writing a cell reference, the column letter should always come first.

Cell reference is very important, especially when writing formulas or addressing the cell to evaluate it. For example, if we want to add values 3 and 2 which are in A1 and A2 respectively, instead of writing A3=3+2, we write A3=A1+A2.

Types of Cell Reference

Relative Cell Reference

A relative cell reference identifies the location of the cell or group of cells. Cell references are used in formulas, functions, charts, and other Excel commands. By default, a spreadsheet cell reference is relative, meaning formulas and functions can be copied and pasted to other cells. A relative cell consists of a column letter and row number, intersecting at the cell location. Examples of relative cell references are A1, B4, G9, etc.

Absolute Cell Reference

An absolute cell reference identifies the location of a cell or group of cells with both the column letter and row number surrounded by dollar signs ($). Examples of absolute cell references are $A$2, $G$9, $Z$345, etc. An easy way to add the dollar sign is to click on the cell reference and then press the “F4” key from the keyboard. Absolute cell references are used when you want a cell reference to stay fixed on a specific cell, meaning that formulas and functions will not change when copied to other cells.

MS-PowerPoint (Presentation)

PowerPoint is the presentation software in the Microsoft Office suite. With PowerPoint, we can create dynamic and professional presentations by using predefined layouts, themes, and templates. PowerPoint includes all the features we need to produce professional-looking presentations. A PowerPoint presentation consists of a series of slides that contain information to communicate with our audience. This information can include text, images, graphs, charts, videos, audio, and more. Presentation software can be used as a supplement to older visual aid technologies such as pamphlets, handouts, posters, and chalkboards.

The extension of PowerPoint files is “.ppt,” while later versions use “.pptx.” Other software alternatives to MS-PowerPoint include MagicPoint, Apple Keynote, IBM Lotus, and Harvard Graphics.

Features of MS-PowerPoint

  • Formatting of data helps to make documents much more attractive, effective, and interactive by using tools like font, font color, hyperlinks, etc.
  • Simple computing operations such as cut, copy, paste, find, and replace are supported.
  • It allows users to use pre-defined layouts, themes, and templates, helping in faster development and design of presentations.
  • Slide transitions and effects can also be used for further requirements.
  • We can create our own custom animations.
  • It supports both internal and external hyperlinks.
  • Different bars, graphs, and charts can be used to convey detailed and precise information.
  • Internet features such as web toolbar, online help, and web linking can also be done.
